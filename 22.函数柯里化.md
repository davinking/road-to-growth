#22.函数柯里化

在数学与计算机科学领域，**柯里化（currying）**是指将接受多个参数的函数变换成以链式方式调用的函数，并且每个链式调用的函数只接受一个参数值。

例如在JavaScript中：
```JavaScript
function multiply(x,y) {
    return x * y;
}

multiply(100,33); // 3300
```
以上函数计算传入的两个数值相乘的结果。

将该函数柯里化：
```JavaScript
function multiply(x) {
    return function(y) {
        return x * y;
    }
}

multiply(100)(33);  // 3300
```
